## List of pages :
### index.html   : 
Acts as homepage, Contains hero section and and blockquote of testimonials
### about.html   : 
Contains multiple articles with aside section that contains the address and a data definition list of acheivements

### contact.html : 
Contains a form with name,email,phone numbers , textarea to enter message. Most importantly it has iframe tag which points the actual location of the office

### services.html: 
Contains multiple ordered and unordered list along a table comparing various services
### team.html    : 
Images are stored in images folder which is displayed along with name,bio and postion

## How to open in your system :

1. Clone this repository
2. Extract all files
3. Open a code editor like (VS Code) to view the code, if extension is downloaded you can host it locally
4. Use navigation bars at the top to navigate across all pages
5. Else you could download and simply click to see the rendered page
